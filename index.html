Banco de dados de poços

O gerenciamento de dados de poços para a empresa de petróleo e gás deve fazer parte de uma visão mais ampla de gerenciamento de dados de subsuperfície para um sistema multidomínio centralizado para gerenciar e armazenar seus dados de subsuperfície em uma única interface. O ciclo de vida do poço envolve muitas funções diferentes em uma organização e várias camadas de sistemas e processos que analisam os dados do poço de diferentes perspectivas e prioridades.
