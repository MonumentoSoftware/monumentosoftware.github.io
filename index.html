Well-X

Todos os dados de poços em um só lugar.

O gerenciamento de dados de poços para empresas de petróleo e gás deve fazer parte de uma visão mais ampla de gerenciamento de dados de subsuperfície. Nosso sistema de gerenciamento de poços Well-X é capaz de processar todos os arquivos de perfis digitais de poços, assim como dados de sísmica de poço, e dados sísmicos em geral. Modelos de de busca com base em perfis similares, clusterização por DBSCAN e várias outras features, totalmente customizáveis.
